**Spotify Analysis**


This project explores listening habits across more than 10,000 Spotify records, using Python to clean, process, and uncover patterns in what people listen to and when. The goal was to take a large, messy dataset of songs and turn it into personalized, meaningful insights about listening trends over time.

Using Pandas and Matplotlib, I built a data pipeline to process the listening records, revealing top artists and genres by year. One of the biggest challenges was inconsistency in genre labeling across more than 15,000 songs, so I applied fuzzy string matching with RapidFuzz to group related genres together and consolidate them into cleaner, more usable categories. From there, I built interactive charts and time-based filters to explore how listening habits shifted across different periods, layering in genre clustering to make the trends easier to interpret.

Once the analysis was complete, I exported the visualizations as PNGs and embedded them into LaTeX-formatted documents, giving the final output a polished, presentation-ready feel. The result was a clear, personalized picture of listening trends — turning a large, unstructured dataset into insights that actually say something about the music someone listens to and how that changes over time.
